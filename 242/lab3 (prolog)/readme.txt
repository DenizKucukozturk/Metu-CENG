Follow the below instructions in order to test your code.
- Download your code and save it to this directory
- Run the grader with the following command;
	swipl -S4096m -q -s section2_grader.pl -t "runGrader('./FILENAME.pl')"
  where FILENAME.pl is your code file.

Note: You may need 4 gbs of memory space to run this command.
If you don't, remove or change the option "-S4096m".
